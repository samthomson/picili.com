picili

picili lets you search and explore pictures you put on dropbox. It synchronises and organises your pictures automatically, making it easy to search and browse your pictures.

It's currently under deveopment, check out the github repo to see how it's coming along. picili on github. Star

Previous Next

How it works

  1. sign in with dropbox and choose a folder
  2. picili will scan it for images and syncronise for any new images or removed images
  3. picili will analyse each picture it finds and add as many tags as it can

Features

  • subject search
  • map search
  • calendar search - day / week / month
  • browse by folder
